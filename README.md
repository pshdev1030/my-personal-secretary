## My Personal Secretery
(link)[http://172.31.45.159:80]

## Describe

Web app where you can manage your schedule and receive information from your ai secretary.

## Development Period

1.11-1.25

## BackEnd

javascript + nodejs + express + mongoose

```
npm i
npm run dev
```


dev

build and run
